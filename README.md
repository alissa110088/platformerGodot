# 🎮 Platformer Project

A 2D platformer created in **1 week using Godot Engine and GDScript**.

## 🎮 About the Game

This project is a small platformer where the player navigates through levels filled with jumps, obstacles, and basic platforming challenges.

It was developed as a rapid prototype to explore movement, level design, and core platformer mechanics in Godot.

## 🧠 Focus

The main focus of this project was:
- responsive player movement
- tight platforming controls
- simple but readable level design

## 🛠️ Features

- 2D platformer movement (run, jump)
- Basic physics interactions
- Level-based progression
- Designed using simple modular mechanics

## ⚙️ Built With

- Godot Engine
- GDScript

## ⏱️ Development

- Developed in **1 week**
- Focused on learning Godot and improving gameplay feel

## 🎯 Goal of the Project

The goal was to understand the fundamentals of 2D platformer design and implement smooth, responsive controls using GDScript.
